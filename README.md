# 💬 FullStack Messaging App

A modern real-time messaging application built with the **MERN Stack**. It provides secure authentication, real-time messaging, online user status, and a responsive user interface.

---

## 🚀 Features

- 🔐 User Authentication (JWT)
- 💬 Real-time Messaging
- 🟢 Online/Offline User Status
- 👤 User Profile Management
- 🎨 Multiple Themes (DaisyUI)
- 📱 Responsive UI
- 🔒 Protected Routes
- ⚡ Fast and Modern Interface
- 🌐 REST API Integration
- ☁️ MongoDB Database

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- DaisyUI
- Axios
- Zustand
- React Router DOM
- Socket.io Client

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Socket.io
- Cloudinary
- BcryptJS
- Cookie Parser

## ⚙️ Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Jayasankar-Bollam/FullStack-MessagingApp.git
```

### 2️⃣ Open Project

```bash
cd FullStack-MessagingApp
```

---

## 📦 Backend Setup

```bash
cd backend
npm install
npm run dev
```

---

## 💻 Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

---

## 🌍 Environment Variables

Create a `.env` file inside the **backend** folder.

Example:

```env
MONGODB_URI=your_mongodb_uri
JWT_SECRET=your_secret_key
PORT=5001
CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret
NODE_ENV=development
```
